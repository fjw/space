Project SPACE - ultraalpha Version

© 2013-2014 Frédéric Worm


This game needs a fast browser preferable on a machine with a good GPU. Use Chrome or Firefox.


up, down - accelerate, brake

left, right - rotate

N - brake to stop

space - blast

B - bomb


maximum viewport

(selected configuration will be saved in a cookie)

I developed this game just for fun. Main problem is that the actual browsers are still not fast enough to handle a fast realtime multiplayer action game like this.
I made several benchmarks, performance tests and rewrote the entire game several times to make it faster.

Also, if I get enough donations, I will release the sourcecode on github.
